|---|---|---|
| Nav label | `Industries` | `Startups` |
| Top-level categories | 20 industries | 6 startup dimensions |
| Subcategories | ~200 | 60 |
| URL base | `/industries/` | `/startups/` |
| Page template | Same | Same — no UI change needed |
| Hover interaction | Same | Same |

The existing `/industries/` pages don't need to be deleted immediately — you can redirect them or leave them for SEO while the new `/startups/` structure builds authority.
